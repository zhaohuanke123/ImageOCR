# OCR 复核工具竞品分析调研报告

**调研日期**: 2026-04-08
**调研目标**: 分析主流OCR标注/复核工具，为本项目（OCR思维导图复核工具）提供设计参考

---

## 一、调研工具概述

### 1. Label Studio

#### 工具简介
Label Studio 是一个开源的多类型数据标注平台，由 Heartex 开发维护。支持图像、音频、视频、文本等多种数据类型的标注，广泛应用于机器学习数据准备。

#### 核心功能
| 功能类别 | 具体能力 |
|---------|---------|
| **标注类型** | 边界框(Bounding Box)、多边形、关键点、语义分割、时间序列标注 |
| **OCR支持** | 支持文本区域标注、OCR转录校正、文本分类 |
| **数据格式** | 支持导入/导出 JSON、XML、CSV、COCO、Pascal VOC 等格式 |
| **协作功能** | 多用户协作、任务分配、标注质量控制 |
| **自动化** | ML Backend 集成，支持预标注模型 |

#### UI/UX 特点
- **界面布局**: 三栏式设计（左侧任务列表、中间标注画布、右侧标注属性面板）
- **标注画布**: 支持缩放（Ctrl+滚轮）、平移（拖拽空处）、框选创建
- **实时预览**: 标注结果即时渲染，支持透明度调节
- **快捷键**: 完善的键盘快捷键支持（数字键切换标签、Delete删除等）

#### 可借鉴设计模式
```
+------------------+------------------------+------------------+
|   任务列表        |      标注画布           |    属性面板      |
| - 待标注          |  [图像/区域]            | - 标签选择       |
| - 已标注          |  [边界框叠加层]         | - 属性编辑       |
| - 有问题          |  [缩放/平移控制]        | - 历史记录       |
+------------------+------------------------+------------------+
```

### 2. CVAT (Computer Vision Annotation Tool)

#### 工具简介
CVAT 是由 Intel 开发的开源计算机视觉标注工具，专注于视频和图像标注。现由 CVAT.ai 团队维护，提供云端和本地部署两种版本。

#### 核心功能
| 功能类别 | 具体能力 |
|---------|---------|
| **标注类型** | 边界框、多边形、折线、关键点、立方体(3D) |
| **视频标注** | 视频帧间标注追踪、插值、时间线编辑 |
| **OCR相关** | 文本区域标注、区域转录、多语言支持 |
| **AI辅助** | SAM 集成、自动分割、跟踪算法 |
| **导入导出** | COCO、VOC、YOLO、MOT、COCO Keypoint 格式 |

#### UI/UX 特点
- **界面布局**: 专业级工具栏布局，画布为中心
- **交互设计**:
  - 框选: 拖拽创建矩形框
  - 缩放: 鼠标滚轮 + 中心点缩放
  - 平移: 中键拖拽或 Space+左键拖拽
  - 调整: 角点/边线拖拽调整大小
- **图像调整**: 支持亮度/对比度调节（右侧面板）
- **图层系统**: 标注对象分层显示，可单独隐藏/锁定

#### 可借鉴设计模式
```
+------------------------------------------------------------+
|  工具栏: [选择] [框选] [多边形] [画笔] [缩放] [撤销] [重做]  |
+------------------------------------------------------------+
|                    |                                        |
|   对象列表         |         标注画布                        |
|   ├─ 对象1         |   [缩放比例指示器]                     |
|   ├─ 对象2         |   [坐标显示]                           |
|   └─ 对象3         |   [标注框+标签]                        |
|                    |                                        |
+--------------------+----------------------------------------+
|   属性面板: 图像调整 | 标签属性 | 历史记录                    |
+------------------------------------------------------------+
```

### 3. LabelImg

#### 工具简介
LabelImg 是一个轻量级的图形化图像标注工具，基于 PyQt 开发。专注于边界框标注，支持 Pascal VOC 和 YOLO 格式输出，适合快速标注任务。

#### 核心功能
| 功能类别 | 具体能力 |
|---------|---------|
| **标注类型** | 仅支持边界框(Rectangle) |
| **格式支持** | Pascal VOC XML、YOLO TXT、CreateML JSON |
| **OCR适用** | 可用于标注文本区域，但不支持文本转录 |
| **批量操作** | 支持目录批量处理、自动保存 |
| **快捷操作** | 极简键盘操作（W创建框、D下一张、A上一张） |

#### UI/UX 特点
- **界面布局**: 极简单窗口设计
- **交互设计**:
  - 框选: 点击拖拽创建，支持微调
  - 缩放: Ctrl+滚轮
  - 平移: 滚轮拖拽
- **实时预览**: 框选时显示实时尺寸
- **标签输入**: 弹窗式标签输入

#### 可借鉴设计模式
```
+------------------------------------------------------------+
|  文件 | 编辑 | 视图 | 帮助                                  |
+------------------------------------------------------------+
|  [打开] [打开目录] [保存] [下一张] [上一张] [验证]            |
+------------------------------------------------------------+
|   文件列表     |              标注画布                      |
|   ├─ img1.jpg |                                            |
|   ├─ img2.jpg |   [图像显示]                               |
|   └─ img3.jpg |   [边界框叠加]                             |
|                |                                            |
+----------------+-------------------------------------------+
|   标注列表     |   标签列表                                 |
|   ├─ box1     |   ├─ person                               |
|   └─ box2     |   └─ car                                  |
+------------------------------------------------------------+
```

## 二、图片标注工具交互模式分析

### 1. 框选交互模式

| 模式 | 描述 | 适用场景 |
|-----|------|---------|
| **拖拽创建** | 从一点拖拽到另一点创建矩形 | 新建标注 |
| **点击调整** | 拖拽角点/边线调整大小 | 精细调整 |
| **整体移动** | 拖拽框体中心移动位置 | 位置校正 |
| **键盘微调** | 方向键移动选中的框 | 像素级调整 |
| **复制粘贴** | Ctrl+C/V 复制标注框 | 相似区域快速标注 |

### 2. 缩放交互模式

| 操作方式 | 实现方式 | 用户习惯 |
|---------|---------|---------|
| **滚轮缩放** | 鼠标滚轮上下 | 最常用方式 |
| **中心缩放** | 以鼠标位置为中心缩放 | 精确定位时更直观 |
| **按钮缩放** | +/- 按钮 | 触摸屏或不习惯滚轮用户 |
| **框选放大** | 框选区域放大到视图 | 快速定位细节区域 |
| **适应窗口** | 自动缩放适应画布 | 查看全貌 |
| **实际大小** | 100% 显示 | 检查细节 |

### 3. 平移交互模式

| 操作方式 | 实现方式 | 备注 |
|---------|---------|------|
| **中键拖拽** | 鼠标中键按住拖动 | CAD软件习惯 |
| **空格+拖拽** | Space 键 + 左键拖动 | PS/Illustrator 习惯 |
| **边框拖拽** | 拖拽画布边缘 | 较少使用 |
| **方向键** | 键盘方向键平移 | 小幅度移动 |
| **滚动条** | 使用滚动条 | 传统方式 |

## 三、图像调整工具 UI 设计

### 1. 常见参数调节控件

| 参数 | 控件类型 | 范围 | 默认值 |
|-----|---------|------|-------|
| **亮度** | 滑块 | -100 ~ +100 | 0 |
| **对比度** | 滑块 | -100 ~ +100 | 0 |
| **饱和度** | 滑块 | -100 ~ +100 | 0 |
| **锐化** | 滑块 | 0 ~ 100 | 0 |
| **色温** | 滑块 | 冷 <-> 暖 | 中间 |
| **缩放比例** | 下拉框/滑块 | 10% ~ 500% | 适应窗口 |

### 2. UI 布局模式

**模式 A: 侧边面板**
```
+------------------+
| 图像调整         |
+------------------+
| 亮度    [====|   ] 0
| 对比度  [|====   ] 0
| 饱和度  [==|     ] 0
|                  |
| [重置] [应用到所有]
+------------------+
```

**模式 B: 浮动工具栏**
```
[亮度: -] [====|====] [+]  [对比度: -] [====|====] [+]  [重置]
```

**模式 C: 弹出面板**
```
点击按钮 -> 弹出面板 -> 调节参数 -> 应用/取消
```

### 3. 实时预览实现

- **CSS Filter 方案**: 使用 `filter: brightness() contrast() saturate()`
- **Canvas 方案**: 使用 `CanvasRenderingContext2D.filter` 或像素级操作
- **WebGL 方案**: 使用着色器实现高性能调整

## 四、关键 UX 模式总结

### 1. 框选交互最佳实践

```
创建流程:
点击起点 -> 拖拽 -> 松开终点 -> 弹出标签输入/自动分配标签

调整流程:
选中框 -> 显示控制点 -> 拖拽控制点/边/整体 -> 实时更新

删除流程:
选中框 -> 按 Delete 键 -> 删除确认(可选)
```

### 2. 实时预览设计

| 预览类型 | 实现方式 | 性能考量 |
|---------|---------|---------|
| **框选预览** | SVG/Canvas 叠加层 | 轻量，推荐 |
| **图像调整预览** | CSS Filter | GPU 加速，推荐 |
| **缩放预览** | CSS Transform | 已优化，推荐 |
| **标注叠加** | SVG 路径 | 适合复杂形状 |

### 3. 结果展示设计

| 信息类型 | 展示方式 | 位置 |
|---------|---------|------|
| **边界框** | 彩色矩形叠加 | 画布上 |
| **标签文本** | 框内/框外标签 | 框附近 |
| **置信度** | 进度条/百分比 | 属性面板 |
| **问题标记** | 图标/颜色高亮 | 列表+画布 |
| **统计数据** | 汇总信息 | 顶部/底部 |

## 五、针对本项目的具体设计建议

### 项目背景
本项目是 OCR 思维导图复核工具，需要处理大尺寸思维导图图像，复核 OCR 识别结果，包括：
- 文本块位置校验
- 文本内容校正
- 层级关系（边）校验

### 设计建议

#### 1. 界面布局建议

```
+------------------------------------------------------------+
|  [菜单栏] 文件 | 视图 | 工具 | 帮助                          |
+------------------------------------------------------------+
|  [工具栏] [选择] [框选] [平移] [缩放] [撤销] [重做] [设置]    |
+------------------------------------------------------------+
|        |                                   |                |
| 节点   |         标注画布                   |   属性面板     |
| 列表   |                                   |                |
|        |   [大图显示区域]                   |   - 文本内容   |
| [全部] |   [节点框叠加]                     |   - 置信度     |
| [问题] |   [边连线显示]                     |   - 父节点     |
| [孤立] |   [缩放比例: 100%]                 |   - 子节点     |
|        |                                   |                |
|        |   [小地图导航]                     |   [图像调整]   |
|        |                                   |   亮度 [===]   |
|        |                                   |   对比度 [===] |
+--------+-----------------------------------+----------------+
|  [状态栏] 节点: 156 | 边: 142 | 问题: 5 | 缩放: 100%       |
+------------------------------------------------------------+
```

#### 2. 核心交互设计

**大图浏览**:
- 由于思维导图图像通常很大，需要实现高效的分块加载
- 使用小地图导航，快速定位区域
- 支持框选区域放大

**框选与编辑**:
- 支持创建新节点（框选文本区域）
- 支持编辑已有节点（调整边界、修改文本）
- 支持删除节点/边
- 支持创建/删除父子关系边

**快捷键设计**:
| 快捷键 | 功能 |
|-------|------|
| V | 选择工具 |
| B | 框选工具 |
| H | 平移工具 |
| Z | 缩放工具 |
| Ctrl+滚轮 | 缩放 |
| Space+拖拽 | 平移 |
| Delete | 删除选中 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Tab | 切换到下一个问题节点 |

#### 3. 图像调整功能

针对 OCR 场景，图像调整主要帮助：
- 提高对比度使文本更清晰
- 调整亮度处理过暗/过亮区域

**推荐实现**:
```javascript
// CSS Filter 方案（实时预览）
canvas.style.filter = `brightness(${brightness}%) contrast(${contrast}%)`;

// 或 WebGL 方案（更好的性能）
gl.uniform1f(brightnessLocation, brightness / 100);
gl.uniform1f(contrastLocation, contrast / 100);
```

#### 4. 问题检测与展示

**问题类型**:
| 问题类型 | 视觉标记 | 处理建议 |
|---------|---------|---------|
| 低置信度节点 | 黄色边框 | 人工复核文本 |
| 孤立节点 | 红色边框 | 创建/删除边 |
| 超大节点 | 橙色边框 | 拆分节点 |
| 弱边关系 | 虚线显示 | 确认/删除关系 |

**问题列表面板**:
```
+------------------+
| 问题列表 (5)     |
+------------------+
| [!] 低置信度 x3  |
| [?] 孤立节点 x2  |
+------------------+
| - 节点#45 (0.42) |
| - 节点#89 (无父) |
| - 节点#102(0.38) |
+------------------+
```

#### 5. 技术选型建议

| 功能模块 | 推荐技术 | 理由 |
|---------|---------|------|
| **画布渲染** | Fabric.js 或 Konva.js | 支持对象选择、变换、事件 |
| **大图处理** | OpenSeadragon 或 Leaflet | 分块加载，支持超大图 |
| **状态管理** | Zustand 或 Jotai | 轻量，适合复杂状态 |
| **快捷键** | hotkeys-js | 成熟的快捷键库 |
| **撤销/重做** | 自实现 Command Pattern | 精细控制 |

## 六、总结

### 工具对比矩阵

| 特性 | Label Studio | CVAT | LabelImg | 本项目需求 |
|-----|-------------|------|----------|-----------|
| **开源** | ✅ | ✅ | ✅ | ✅ |
| **OCR支持** | ✅ | ✅ | ⚠️ | ✅ |
| **大图支持** | ⚠️ | ✅ | ❌ | ✅ |
| **图像调整** | ❌ | ✅ | ❌ | ✅ |
| **关系标注** | ❌ | ⚠️ | ❌ | ✅ |
| **实时预览** | ✅ | ✅ | ✅ | ✅ |
| **部署复杂度** | 中 | 高 | 低 | 低 |

### 核心借鉴点

1. **从 CVAT 借鉴**:
   - 专业级缩放/平移交互
   - 图像调整面板设计
   - 对象列表管理

2. **从 Label Studio 借鉴**:
   - 三栏式布局
   - 标签/属性面板设计
   - 任务流程管理

3. **从 LabelImg 借鉴**:
   - 极简快捷键设计
   - 快速标注流程
   - 轻量级部署

### 创新点建议

1. **思维导图特化**:
   - 自动布局预览
   - 层级关系可视化
   - 节点合并/拆分工具

2. **OCR 复核特化**:
   - 原文 vs 识别文本对照
   - 置信度热力图显示
   - 批量校正建议

3. **协作功能**:
   - 多人实时协作（可选）
   - 审核流程支持
   - 变更历史追踪
